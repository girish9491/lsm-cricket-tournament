## Troubleshooting

### "Edit button not showing"
- Check if admin has locked editing
- Only captains and admin can edit teams
- Verify you're logged in correctly

### "Cannot edit team"
- Make sure edit is unlocked (check with admin)
- Enter correct captain mobile number
- You must be the registered captain

### "No teams in Pool Management"
- Wait for page to load completely
- Refresh the page
- Check if teams are registered in the system

### "Fixtures not showing"
- Admin must release fixtures first
- Check Pool Management section
- Ensure pools are saved
