# Background Image Instructions

## Login Background Image

Place your 4K cricket stadium background image here with the name:
**login-bg.jpg**

### Image Requirements:
- **Name**: login-bg.jpg (exactly)
- **Format**: JPG, JPEG, or PNG
- **Recommended Resolution**: 1920x1080 (Full HD) or 3840x2160 (4K)
- **Aspect Ratio**: 16:9
- **File Size**: Under 3MB for fast loading

### Where to Find Cricket Stadium Images:
1. **Unsplash**: https://unsplash.com/s/photos/cricket-stadium
2. **Pexels**: https://www.pexels.com/search/cricket%20stadium/
3. **Pixabay**: https://pixabay.com/images/search/cricket-stadium/

### How to Add:
1. Download a high-quality cricket stadium image
2. Rename it to: **login-bg.jpg**
3. Save it in this folder: `LSM-Cricket-Tournament/images/`
4. Redeploy to Netlify

The background will automatically appear on the login page!

### Current Setting:
The CSS already points to: `images/login-bg.jpg`

If no image is found, the page will show a solid color background.
