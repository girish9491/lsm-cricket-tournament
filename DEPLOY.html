üî• Setup Firebase Database (10 Minutes)

‚ö†Ô∏è Important: Without Firebase setup, data will only save in the browser. Set up Firebase so all team registrations are saved in the cloud!
1 Create Firebase Project
  • Go to: Firebase Console
  • Click "Add Project"
  • Name: "LSM Cricket Tournament"
  • Disable Google Analytics
  • Click "Create Project"
2 Enable Realtime Database
  • Click "Realtime Database" from left menu
  • Click "Create Database"
  • Choose location closest to India
  • Important: Select "Start in TEST MODE"
  • Click "Enable"
3 Set Database Rules
  • Click "Rules" tab
  • Replace with this code:
{ "rules": { "teams": { ".read": true, ".write": true } } }
  • Click "Publish"
4 Get Firebase Config
  • Click ‚öôÔ∏è (Settings) > Project Settings
  • Scroll to "Your apps"
  • Click Web icon ()
  • App name: "LSM Cricket App"
  • Copy the firebaseConfig object
5 Update script.js
  • Open script.js file
  • Find lines 2-10 (firebaseConfig section)
  • Replace with YOUR Firebase config
  • Save the file
6 Re-deploy to Netlify
  • Go back to Netlify dashboard
  • Click your site
  • Drag the updated folder again to deploy
  • Or click "Deploys" > "Drag and drop your site folder here"
